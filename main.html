How to play

  1. Help Naruto get through the maze to find Sasuke.
  2. Build a computer program by dragging code blocks from the center and clicking them together in the space on the right.
  3. Click “Run” to see what your program does.
  4. Try to solve the puzzle using the least number of code blocks.

Help with Code Blocks

Code blocks are instructions you give to the computer. Click them together under the “When run” block to create a program to control Naruto. Here’s how they work:

When you click the “Run” button, everything connected to this block will be done in order.

Moves Naruto one square in the direction it is pointing.

Turns Naruto to its right (from the Naruto’s perspective).

Turns Naruto to its left (from the Naruto’s perspective).

Checks the color of the square Naruto is on. If it’s blue, then any blocks connected inside this one will be done in order. Otherwise they will be skipped.

Checks the color of the square Naruto is on. If it’s green, then any blocks connected inside this one will be done in order. Otherwise they will be skipped.

Runs all the blocks inside a specific number of times (you can change the number)

Runs all the blocks inside over and over until Naruto found Sasuke or Naruto gets stuck.
